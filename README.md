### Usage: mhgu-charm-export [OPTIONS] <FILE_PATH>
```
Arguments:
  <FILE_PATH>  

Options:
  -o, --out <OUT>    
  -s, --slot <SLOT>  
  -h, --help         Print help
  -V, --version      Print version`
```

build from source:
> cargo build --release

run from source:
> cargo run <your_args_here>
